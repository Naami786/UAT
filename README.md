# UAT
是一个Web版的自动化用例及测试任务管理平台，底层基于Robotframework去执行生成的测试脚本。

借助Chrome插件，及H5页面的特性。使您可以通过轻松的点点点，即可生成测试脚本，可视化的元素标签及关键词释义，使你的用例维护更便捷。
